## Conventions across SDKs

- Every SDK accepts `{ apiKey, baseUrl, timeout, maxRetries }`.
- All methods return `{ data, raw, request_id }`. The raw response is there
  for debugging.
- Errors subclass a shared `NexaError`. Switch on `error.type` to branch.
- Automatic retries on `5xx` and `429` with exponential backoff.
