# NexaCreate API

<span className="nexa-product-pill" data-product="create">NexaCreate</span>

NexaCreate generates images and videos from prompts, references, or templates.
The hard parts — model selection, queueing, credit accounting, webhook signing —
are handled for you. You talk to one surface.

## Model routing

You never pick a raw model id. You pick a **preset** and NexaCreate routes to
the best-available backend (Replicate, Runway, Pika, Fal, …) based on price,
latency and quality score.

| Preset                     | Use for                                    |
| -------------------------- | ------------------------------------------ |
| `image.fast`               | Thumbnails, drafts                          |
| `image.quality`            | Hero images, marketing visuals              |
| `image.realistic_portrait` | People, skin, hands                         |
| `video.fast`               | 5–10 s social clips                         |
| `video.cinematic`          | Pitch videos, ads                           |
| `video.lipsync`            | Avatars, voice-driven talking heads         |

Override with `model: "replicate/…"` if you need determinism.

## Async by default

Every generation is a job. Poll or subscribe via webhook.

```
POST /v1/images    → 202 + job_id
POST /v1/videos    → 202 + job_id
GET  /v1/jobs/{id} → { status, progress, result? }
```
