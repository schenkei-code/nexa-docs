# Google Tools (35+)

<span className="nexa-product-pill" data-product="ads">NexaAds · MCP</span>

Google-side tools sit under the `nexaads.google.*` namespace. They wrap the
Google Ads API v17 and share NexaAds auth/quota handling.

<Callout type="info">
  The Google surface is planned but pre-registered: calling `GET /mcp` already
  returns these tool names so agents can plan around the full roadmap.
</Callout>
