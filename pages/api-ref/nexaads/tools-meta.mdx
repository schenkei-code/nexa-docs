# Meta Tools (60+)

<span className="nexa-product-pill" data-product="ads">NexaAds · MCP</span>

All Meta-related tools sit under the `nexaads.meta.*` namespace on the
MCP server. They mirror the shape of the underlying Meta Graph API but with
auth, retries, rate-limit smoothing and idempotency handled for you.

<Callout type="info">
  Tool signatures below document the in-repo catalogue from
  `src/lib/mock-data.ts`. The tools marked **live** are wired to
  production; the rest are planned but already discoverable via
  `GET /mcp` so agents can see what's coming.
</Callout>

## Campaigns (live)

### `nexaads.campaigns.list`

List campaigns with filter and sort options.

```json
{
  "platform": "string (optional) — meta | google | tiktok",
  "status":   "string (optional) — Aktiv | Pausiert | Entwurf | Beendet",
  "limit":    "number (optional, default 50)"
}
```

### `nexaads.campaigns.create`

Create a new campaign.

```json
{
  "name":      "string (required)",
  "platform":  "string (required)",
  "budget":    "number (required)",
  "targeting": "object (optional)"
}
```

### `nexaads.campaigns.update`

```json
{ "campaignId": "string (required)", "updates": "object (required)" }
```

### `nexaads.campaigns.pause`

```json
{ "campaignId": "string (required)" }
```

## Reports (live)

### `nexaads.reports.generate`

```json
{
  "dateRange": "{ from: string, to: string } (required)",
  "platforms": "string[] (optional)",
  "metrics":   "string[] (optional)"
}
```

### `nexaads.reports.list`

```json
{ "limit": "number (optional, default 20)" }
```

## Budget (live)

### `nexaads.budget.optimize`

AI-driven reallocation across a set of campaigns.

```json
{
  "campaignIds": "string[] (optional — all if empty)",
  "strategy":    "string (optional) — maximize_roas | minimize_cpa | balanced"
}
```
