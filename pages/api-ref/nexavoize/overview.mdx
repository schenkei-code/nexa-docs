# NexaVoize API

<span className="nexa-product-pill" data-product="voize">NexaVoize</span>

NexaVoize powers real-time voice assistants and high-quality text-to-speech
synthesis. Built on XTTS v2 for speech synthesis and Claude for conversational
memory, with sub-300 ms streaming turn-taking.

## Base URL

```
https://api.nexavoize.com/v1
```

## Common endpoints

| Method | Path                              | Purpose                        |
| ------ | --------------------------------- | ------------------------------ |
| POST   | `/v1/voice/synthesize`            | Render text to audio           |
| POST   | `/v1/voice/synthesize/stream`     | Stream audio as it renders     |
| GET    | `/v1/voices`                      | List available voices          |
| POST   | `/v1/voices`                      | Clone a voice                  |
| POST   | `/v1/conversations`               | Open a live session            |
| POST   | `/v1/conversations/&#123;id&#125;/audio` | Push user audio              |
| GET    | `/v1/assistants`                  | List persistent assistants     |
| POST   | `/v1/assistants`                  | Create assistant with persona  |

## Quickstart

```bash
curl https://api.nexavoize.com/v1/voice/synthesize \
  -H "Authorization: Bearer $NEXA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "The best way to predict the future is to invent it.",
    "voice_id": "nv_classic_male",
    "format": "mp3"
  }' \
  --output speech.mp3
```

## Latency

Streaming synthesis typically returns the first audio chunk in ~180 ms and
keeps up with real time (1× speed). For conversational loops keep the same
WebSocket open instead of reconnecting per turn.
